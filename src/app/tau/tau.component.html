Multiple Server Management

Register multiple servers, select a server from your server list to work with. Custom user settings per server.

Auto Update

Automated version checker, select the perfect TAU version to your server, list the available TAU versions, one click update.

Real-Time Connection

Check your stored data and server status anytime you wish, in real-time.

Flexible Design

Built-in modular and tab oriented design. Split your window and design your custom working environment. Open your required sections in tabs.

Schema-free

Create datasets using schema-free JSON storage engine. Define your dataset by schema or sample document.

Import

Built-in import functions. Import documents and tags from CSV or JSON format. Automated field matching & bulk import. Custom bulk size settings.

Create

One click dataset creation. Auto filled settings, and sample document templates.

Clone

Right-click dataset clone option. Create a new dataset cloning the attributes of an existing one.

Rename

Rename a dataset by right-click option. Get instant access to a dataset, rename it freely.

Remove

Right-click option. Remove a dataset easily. Use dataset for temporary data storing.

Add, List, Edit, Remove

Quick basic actions. Easily create new documents, list and filter them, edit or remove functions. Perfect for manual moderation or checking process.

Dynamic Tag Management

Manage your tags or categories by datasets. Each dataset has a dedicated tag manager. Create or import tags, use built in filters. Create category tree by organizing tags into hierarchy.

Field Filter

List the available fields inside a dataset, select the required ones to display. Great for complex data objects.

Search

Built-in search options. Quick, flexible search queries. Use logical expressions, field filters or wildcards.

Bulk Actions

Get things done faster. Available options: remove, tag management (add, edit, remove), modification.

Sampling

Built-in statistical sampling function. Generate random document samples from the selected dataset. Available options: tag filtering, stratified sampling options, fix or dynamic sampling sizing.

Create

Create data processing service. Select the required type of service, provide the name and description of it. Using a service you can easily create smart solutions to process your data.

List

List and manage your available services. Multiple view options, easy status and alias modification.

Alias

Use Alias to integrate a service into your process. Identify services by alias, update a service and change them by alias.

Classifier Service

Powerful text classification service. Automated workflow, high accuracy, language independent. Specialized for e-commerce.

PRC Service

Keyword extraction and match making service. Use PRC Service to build product recommendation modules, for SEO optimization and automated duplicate checking.

List, Manage

List and manage the background processes. Each long-term action is organized into a background process.

Download Files, Export

Use Alias to integrate a service into your process. Identify services by alias, update a service and change them by alias.

Create as many classifier engines as you wish by a few clicks. Automize your e-commerce classification or moderation process.
