Which Server Size I need?

It depends on the data size you store, service numbers, and request numbers. Generally as more data you wish to manage as bigger server you need, but it strongly depends on the planned usage, required SLA and response time.

When To Purchase On-Premise Server?

You can purchase life-time Slamby license by processor core numbers. After purchasing your licenses our colleagues help you to install your Slamby Server(s) locally on your machines.

One year support is must, but then there is no need to make it longer. Don't forget with a support agreement all the latest features are available.

Perfect option for a big corporation having a private cloud, or high data security policy. Having on-premise Slamby Servers, you can manage your data without having it outside of your company.
